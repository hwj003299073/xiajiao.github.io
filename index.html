一、三类请求如何快速判断（GET / POST / API）

  • GET（静态页面）

    • 请求方式：GET

    • 数据位置：HTML 源码中

    • 判断：F12 → Network → 请求类型 GET；右键查看页面源代码能看到数据

    • 抓取示例：

import requests
res = requests.get('https://example.com/article?id=123')
html = res.text

  • POST（提交表单）

    • 请求方式：POST

    • 数据位置：请求体（form-data 或 payload）

    • 判断：交互（登录、搜索）触发，F12 → Network 显示 POST

    • 抓取示例：

xxxxxxxxxx
payload = {'username': 'stu', 'password': '123456'}
res = requests.post('https://example.com/login', data=payload)

  • API（异步接口）

    • 返回格式：JSON

    • 特征：XHR / Fetch 类型请求、URL 常含 /api/ /ajax/ 等

    • 判断：页面滚动或点击“加载更多”时出现 XHR 请求，响应是 JSON

    • 抓取示例：

xxxxxxxxxx
res = requests.get('https://api.example.com/list?page=2')
data = res.json()

二、抓取流程快速清单（上手步骤）

  1. 打开 DevTools（F12）→ Network，刷新页面，观察请求类型与响应内容（HTML / JSON）。

  2. 确认是 GET / POST / API：按上面分类判断。

  3. 对于需要登录或带 cookie 的页面：先用浏览器登录，复制请求头（headers）与 cookie。

  4. 在代码中使用 requests 模拟，并带上 headers、cookies。

  5. 如果数据在 JS 渲染后才出现，优先找对应的 API 接口。

三、在 PyCharm 中把浏览器复制的 headers 快速转成 Python 字典（正则替换）

1) 浏览器复制的 headers 原始格式示例：

xxxxxxxxxx
User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64)...
Accept: text/html,application/xhtml+xml,application/xml;q=0.9
Cookie: _abc=xxx; _def=yyy

2) PyCharm / VSCode 正则替换设置（步骤）

  • 打开文件/粘贴 headers 文本

  • 打开替换对话框（Ctrl+R 或 Ctrl+H），启用正则（Regex / .*）

  • 查找（Find）：(.*?): (.*)

  • 替换为（Replace）：'$1': '$2',

替换后将得到：

xxxxxxxxxx
'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64)...',
'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9',
'Cookie': '_abc=xxx; _def=yyy',

然后把它们包入 headers = { ... } 即可。

⚠️ 注意：如果 value 本身含有单引号 '，可先将替换模板改为 \"$1\": \"$2\",（双引号）或对内部单引号做转义。

四、正则小技巧（常用抓取与替换）

  • (.*?): (.*)

    • ( ) 是捕获分组，$1 / $2 在替换时分别对应键 / 值。

    • ? 跟在 .* 后代表非贪婪匹配（.*?）与贪婪匹配（.*）的差别。

  • .* vs .*?：

    • .*：贪婪，会匹配尽可能多的字符。

    • .*?：非贪婪，会匹配尽可能少的字符（常用于 HTML 标签匹配）。

  • 示例：

    • 文本：第一个第二个

    • 正则 .* → 匹配整个字符串

    • 正则 .*? → 只匹配第一个 第一个

五、headers 与伪装（常用字段）

xxxxxxxxxx
headers = {
  'User-Agent': 'Mozilla/5.0 ...',
  'Accept-Language': 'zh-CN,zh;q=0.9',
  'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9',
  'Referer': 'https://example.com',
}

  • 建议至少带上 User-Agent 与 Referer，必要时带上完整 Cookie。

六、示例：综合抓取范例

​x
import requests
from bs4 import BeautifulSoup


url = 'https://example.com/articles?page=1'
headers = {
    'User-Agent': 'Mozilla/5.0 ...',
}
res = requests.get(url, headers=headers)
res.encoding = res.apparent_encoding
soup = BeautifulSoup(res.text, 'html.parser')
for item in soup.select('.article-item'):
    title = item.select_one('h2').get_text(strip=True)
    link = item.select_one('a')['href']
    print(title, link)
